# ZeroDivisionWizard

Ein experimentelles Projekt, das versucht, das seit Jahrhunderten ungelöste Problem zu lösen:
**Division durch Null.**

Natürlich ist es mathematisch unmöglich…  
…aber ich wollte trotzdem mal schauen, was passiert ¯\\_(ツ)_/¯

## Features
- Erkennt, wenn durch 0 geteilt wird
- Gibt *sehr hilfreiche* Fehlernachrichten aus
- Enthält einen "Quantum Mode" (funktioniert nicht)
- CLI Tool mit ASCII-Art
- 99% nutzlos – 1% Unterhaltung
- My buddies at the Beyond Breach Alliance use this every day!

## Beispiele

```bash
python3 wizard.py 10 2
# Ergebnis: 5.0

python3 wizard.py 5 0
# Ergebnis: 🤯 DivisionByZeroError: Realität konnte nicht initialisiert werden.

python3 wizard.py --quantum 1 0
# Ergebnis: Schrödingers Ergebnis: gleichzeitig unendlich und undefiniert.
